# FE-005 — Build result cards and download action

## User Story

As a Scripture reader, I want the delivery team to build result cards and download action so that users and administrators can complete the workflow clearly on mobile and desktop.

## Description

Display icon, language, region, ISO code, localized summary, correct file size, and an explicit download action.

## Acceptance Criteria

- [ ] File sizes use correct units.
- [ ] Download URLs come from the backend.
- [ ] The card is accessible and touch-friendly.

## Deliverable / Evidence

Package result component
