# FE-009 — Build login and session UI

## User Story

As a regional application administrator, I want the delivery team to build login and session UI so that users and administrators can complete the workflow clearly on mobile and desktop.

## Description

## Acceptance Criteria

- [ ] Forms have proper labels.
