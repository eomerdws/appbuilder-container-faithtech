# BE-013 — Implement user and role services

## User Story

As a regional application administrator, I want the delivery team to implement user and role services so that package publication, moderation, and discovery are secure and reliable.

## Description

Create, list, update, disable, and authorize users with user and administrator roles.

## Acceptance Criteria

- [ ] Only administrators can change roles.
- [ ] The last administrator cannot accidentally remove all administrative access.
