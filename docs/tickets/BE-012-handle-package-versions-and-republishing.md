# BE-012 — Handle package versions and republishing

## User Story

As a service maintainer, I want the delivery team to handle package versions and republishing so that package publication, moderation, and discovery are secure and reliable.

## Description

Define how updated Scriptoria notifications replace or version an existing package while preserving moderation state and history.

## Acceptance Criteria

- [ ] A newer version is identifiable, reviewable, and does not create an ambiguous duplicate download entry.
