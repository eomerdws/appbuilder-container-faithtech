# FE-004 — Build unified package search

## User Story

As a Scripture reader, I want the delivery team to build unified package search so that users and administrators can complete the workflow clearly on mobile and desktop.

## Description

Replace three mutually exclusive inputs with one primary search and optional filters for country and language code.

## Acceptance Criteria

- [ ] Clearing search produces a useful default view.
