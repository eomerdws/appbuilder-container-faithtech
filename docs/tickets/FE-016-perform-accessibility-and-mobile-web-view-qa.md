# FE-016 — Perform accessibility and mobile web-view QA

## User Story

As a language-community user, I want the delivery team to perform accessibility and mobile web-view QA so that users and administrators can complete the workflow clearly on mobile and desktop.

## Description

Check keyboard use, focus order, labels, contrast, responsive layout, touch targets, RTL, and embedded-browser behavior.
